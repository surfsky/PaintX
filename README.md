# XPaint

尝试实现 Sketch 跨平台版本


# 布局

    顶部：常用操作，如文件、上下移动图层、删除图层、放缩、对齐等
    左侧：页面、面板、层  & 控件
    右侧：属性编辑窗口
    独立出绘图基础类库
    采用跨平台技术（如 dotnet core + gtk#），便于移植

## 键盘

- [ ] ctrl+A 全选
- [ ] 选中图层后，按住shift拖动图层，显示辅助线，并自动吸附
- [ ] 选中图层后，按住shift旋转图层，旋转时限制角度为45度的倍数
- [ ] 选中图层后，按住option/alt键盘拖动图层，复制图层
- [ ] 选中图层后，按 delete 删除选中图层
